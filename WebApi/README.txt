
Steps for compiling and running this project:

1) Load the solution in visual studio 2017, make sure 'WebApi' is startup project.
2) Restore the nuget packages
3) Compile and run the application ( CTRL + F5 )

To run Tests :

1) Navigate to 'Test Explorer' tool in visual studio from 'Test > Windows'
2) It will discover the tests available in the solution and list them available
3) Click 'Run All' from top menu button!!
